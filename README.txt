Ascii Draw

This submission includes a maven project containing source files for the main program and unit tests. 
The program includes Java 8 features. 

To run:
The maven pom.xml allows creation of an executable jar file by invoking the 'package' build phase.
This phase will create a jar file asciidraw-0.0.1-jar-with-dependencies.jar which can then be run with:

java -jar asciidraw-0.0.1-jar-with-dependencies.jar

Also, the program may be invoked simply by running the main class rates.structuring.asciidraw.App
